# Austin 311

**[austin311.com](https://austin311.com)** — Live maps and trend data on what Austin's city government is (and isn't) doing.

Not a government website. Built by a concerned citizen using publicly available open data.

## Data sources

- **[Austin Open311 API](https://311.austintexas.gov/open311/v2)** — all 311 service requests
- **[Austin Open Data Portal](https://data.austintexas.gov)** — APD crime, crashes, restaurant inspections, court data, water quality, childcare, MetroBike
- **[Texas Open Data Portal](https://data.texas.gov)** — TABC mixed beverage sales (bar of the month)

All data is publicly available. Get your own API key at [data.austintexas.gov](https://data.austintexas.gov).

## How it works

Static HTML maps generated by Python scripts, deployed to GitHub Pages at `docs/`. GitHub Actions workflows regenerate the maps on a daily/weekly schedule and commit the output back to the repo.

```
scripts/generate_map.py <category>   # regenerate any map locally
python scripts/generate_map.py crime-trends
```

The repo also contains a Telegram bot (`austin311_bot.py`) that exposes the same data via [@austin311bot](https://t.me/austin311bot), currently in maintenance mode.

### Local setup

```bash
python -m venv .venv && source .venv/bin/activate
pip install -e .
cp .env.example .env   # add AUSTINAPIKEY (optional, for Socrata rate limits)
python scripts/generate_map.py graffiti
```

Generated maps are written to `docs/<category>/index.html`.

### Environment variables

| Variable | Required | Description |
|----------|----------|-------------|
| `AUSTINAPIKEY` | No | Socrata app token — higher rate limits for crime, budget, court, restaurant, water pages |
| `TELEGRAM_BOT_TOKEN` | Bot only | Telegram bot token from [@BotFather](https://t.me/BotFather) |
