What TERM / ACT / IN actually mean: These are workflow pipeline statuses — not verdicts. TERM (Terminated) = case closed, but the dataset doesn't record how it closed (guilty plea, dismissal, acquittal, etc.). ACT (Active) = open, in progress. IN (Inactive) = open but dormant — stalled, unserved warrant, or awaiting action.

Austin does not publish conviction or dismissal data for general court caseloads. The is the only charge category on this portal where full disposition data is available.
About Prop B disposition data: This dataset covers Proposition B (camping ban) offenses filed at Austin Municipal Court and Downtown Community Court. It is the only charge type on data.austintexas.gov where actual verdict/dismissal data is published.

Charge types cover multiple ordinances: "Camping in a Public Area" (city code), "Camping – State Law" (Tex. Penal Code §48.05, added 2023), and "Sitting, Lying Down, Sleeping Outdoors."
