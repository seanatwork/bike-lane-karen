What this data covers: Travis County District Courts handle felonies and Class A/B misdemeanors (serious assaults, DWI, drug trafficking, etc.). County Courts at Law handle Class A/B misdemeanors and civil matters. This is not Austin Municipal Court — those are lower-level Class C misdemeanors (traffic, parking, noise).

Cases filed = new cases entering the system each period. Cases disposed = cases resolved (by any means — guilty plea, dismissal, trial verdict, transfer). Backlog = total cases still pending at period end. When filed > disposed over time, backlog grows.

Source: Texas Office of Court Administration (OCA) via data.texas.gov. Data does not include disposition type — only volume. Data available through December 2025; updated monthly.
